# Sensor-de-humedad-con-Arduino
Código de Arduino para leer la humedad y temperatura con un DHT11, enseñar los valores en la LCD e indicar si hace falta agua con un LED


Se requieren librerías para la LCD y el sensor de humedad DHT11

Sensor de humedad DHT: https://github.com/adafruit/DHT-sensor-library.git

Pantalla LCD: https://github.com/arduino-libraries/LiquidCrystal.git
